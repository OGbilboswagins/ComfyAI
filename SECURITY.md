# Security Policy

## Supported Versions

Only the latest version of **ComfyAI** is actively supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability:

1. **Do not** open a public issue.
2. Contact the maintainer privately via GitHub (Security Advisories or direct contact).
3. Provide as much detail as possible (steps to reproduce, logs, potential impact).

We will:

- Acknowledge your report as soon as possible.
- Investigate and confirm the issue.
- Work on a fix and coordinate disclosure if needed.

Thank you for helping keep ComfyAI and its users safe.
